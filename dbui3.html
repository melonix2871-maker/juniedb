GenosDB + GenosRTC · Superadmin Console · SaaS/API-ready skeleton

Modern, simple admin GUI for a P2P database

This is a single-file index.html console built on top of GenosDB (browser-first P2P graph DB) and GenosRTC (WebRTC layer). It includes:

🔐
AuthenticationSuperadmin login (default demo/demo) with in-app credential settings.
🧩
SaaS-readyTenant config (db name, RTC, relays, encryption password) stored locally, exportable.
🔑
API-readyAPI keys manager + an in-page API emulator to prove key validation + operations.
🕸️
GenosRTCPeer list + realtime messages via data channels (no backend).
Notes: This is a GUI + reference implementation. Real SaaS production use typically adds a backend for billing, org management, key rotation, audit retention, and server-side API gateway. GenosDB itself can run purely client-side via CDN and supports rtc: true.
